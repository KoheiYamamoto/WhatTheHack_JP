# What The Hackとは?

- 課題解決型ハッカソン
- 3~5人でチームとなり、与えられた技術課題を解決する
- ステップバイステップをなぞるのではなく、チームで解決方法を探し出す
- お互いの学び、知識の共有が重要
- コーチは答えを出さない、ヒントはあり

Master: https://github.com/Microsoft/WhatTheHack
